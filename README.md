# 🐍 Python Password Manager
A simple password manager made with Python and TKinter

## ⚡ Features
- **Search Function**. If you have existing entries, you can type the name of the website and it will return the credentials you've entered.
- **Password Generator**. For convinience, it generates strong and randomize passwords upto 18 characters in total.
- **JSON File**. It provides a way to save your passwords. It works hand-in-hand with Search Function.
- **TKinter**. This allows the user to interact with the program via Graphical User Interface (GUI)

## 📌 Python Concepts Applied
- Exception Handling
- Creating Functions
- Importing Libraries
- Conditional Statements
- TKinter
- Read and Write

## 🚀 Runing the Program
To run the project in your local environment, follow these steps.
1. Download all the files.
2. Choose your own IDE.
3. All files and sub directories must be in the same folder.
4. Install all required libraries. Check main.py for the required libraries.
5. Run main.py

Note: The JSON file in output directory is a sample.
